## Koa
### 中间件
 * koa-static-cache: 静态文件代理服务
 ```
 npm install koa-static-cache
 ```
 * koa-router: 路由
 ```
 npm install koa-router
 ```
 * koa-swig: 模板引擎
 ```
 npm install koa-swig
 ```
 * koa-bodyparser:body解析,数据提交
 ```
 npm install koa-bodyparser
 ```
 * koa-multer: formData解析,文件上传
 ```
 npm install koa-multer
 ```
